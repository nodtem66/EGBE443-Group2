Editor's Note

ธนวินท์ ตระกูลวิไลวรรณ ()

การเรียนการสอนวิชา Image Processing นี้เน้นการเรียนแบบ self-study โดยให้นักศึกษาทำการค้นคว้า และสอบถามอาจารย์หรือ TA หากมีข้อสงสัย จากนั้นทำ project ทุกๆสัปดาห์โดยใช้ความรู้ในเรื่องนั้น นำมาแลกเปลี่ยนเทคนิคกันแต่ละกลุ่ม ทำให้นอกจากจะได้พัฒนาทักษะในการเขียนโปรแกรม เรียนรู้เทคนิคการทำ image processing แบบต่างๆแล้ว ยังได้ทักษะการศึกษาค้นคว้าด้วยตนเองอีกด้วย ซึ่งเป็นประโยน์อย่างมาก เพราะเมื่อเรียนสูงขึ้นไปแล้วไม่มีใครจะมาคอยนั่งสอนให้ตลอด การเรียนด้วยตนเองจึงมีความสำคัญมาก อีกสิ่งที่ได้จากการเรียนวิชานี้ คือ ทักษะการทำงานแบบเป็นทีม เพราะในแต่ละสัปดาห์จะมีการศึกษาเทคนิคที่ใช้ในแต่ละบท หลากหลายเทคนิค จึงมีการแบ่งงานกัน และมารวมโค้ดโปรแกรมกัน หรือแม้กระทั่งในบทแรกๆที่เรียนรู้เกี่ยวกับการทำกล้องรูเข็ม ก็ได้เรียนรู้หลักการ เทคนิค การทำงานของกล้องและเทคนิคการถ่ายภาพแบบต่างๆ

จงสุข สงวนตระกูล ()

เป็นการเรียนการสอนแบบใหม่ที่อาจารย์นำมาใช้กับชั้นเรียน โดยจะเน้นไปที่การเรียนรู้ด้วยตัวเองและการทำโปรเจคส่งในแต่ละสัปดาห์ ส่งผลให้นักศึกษาได้ฝึกการค้นคว้านอกห้องเรียน การสอบถามจากผู้รู้ เช่น อาจารย์ TA รุ่นพี่ต่างๆ หรือเพื่อนต่างกลุ่มในชั้นเรียน เป็นต้น และฝึกให้มีความรับผิดชอบ เพราะทุกสัปดาห์จะมีเส้นตายในการกำหนดส่งงาน อีกทั้งการทำโปรเจคของ Image Processing จะให้ทำงานใน matlab ซึ่งนักศึกษาจะได้ฝึกการเขียนโปรแกรมโดยตรง และการทำงานเป็นกลุ่มที่ต้องแลกเปลี่ยนความรู้กัน ทำงานไปด้วยกันช่วยเหลือกัน วิชานี้จึงสอนอะไรหลายๆอย่างที่ไม่มีในห้องเรียน ทำให้นักศึกษาได้พัฒนาตนเองไปหลายขั้น แล้วในวิชานี้ส่วนแรกๆยังมีการเรียนรู้เกี่ยวกับกล้องรูเข็ม การเลือกซื้อกล้องและอุปกรณ์หลายๆ และเทคนิคการใช้กล้องและถ่ายภาพแบบโปร จึงเรียกได้ว่า”วิชานี้สุดยอดจริงๆ ให้ทั้งการเรียนรู้บทตำราและชีวิตจริง สุดท้ายผมไม่รู้ว่าวิชาอื่นจะดีกว่าอย่างไร แต่ถ้าใครมาถามผม ผมจะแนะนำให้ลงเรียนวิชานี้ Image Processing”

ไชยกร ทองชนะสิน ()

โดยทั่วไปวิชา Image Processing ก็จะเรียนเกี่ยวกับการประมวลภาพตามชื่อวิชา แต่ Image Processing ของชั้นเรียนนี้สอนตั้งแต่การสร้างภาพ ทำให้ได้รู้จักองค์ประกอบต่างๆของภาพ เมื่อได้มาเริ่มขั้นตอนการดำเนินการกับภาพ ก็ทำให้เข้าใจวิธีการและความหมายของการดำเนินการ จุดประสงค์ของแต่ละวิธีการทำก็แสดงออกมาด้วยการทำโปรเจคที่จะมีกำหนดส่งในแต่ะสัปดาห์ ซึ่งในระหว่างชั่วโมงเรียนวิชานี้ ก็จะเป็นการเรียน การค้นคว้าด้วยตัวเอง ย่อมต้องเกิดคำถาม หรือข้อสงสัยในการทำ อาจารย์และพี่ๆTA ก็จะช่วยตอบคำถามเหล่านั้น เป็นการทำให้เกิดการเรียนรู้และพัฒนาไปพร้อมๆกันของผู้เรียนแต่ละคน และการแบ่งกลุ่มผู้เรียน ก็เป็นตัวกระตุ้นกลุ่มผู้เรียนให้มีทั้งการวางแผน การทำงานร่วมกัน และการแลกเปลี่ยนความรู้ระหว่างกลุ่ม จึงต้องขอขอบคุณอาจารย์ พี่ๆTA และเพื่อนๆ ในการเรียนและการทำโปรเจค เพราะเป็นการเรียนการสอนแบบใหม่ที่ให้สิ่งต่างๆที่อยู่ทั้งในและนอกห้องเรียนจริงๆ

ธัญพร พึ่งรัศมี ()

สำหรับวิชา image processing ต้องบอกก่อนเลยว่าตอนแรกเป็นคนที่ไม่ชอบการประมวลผลทางภาพเลย อาจจะเป็นเพราะว่าไม่เคยรู้มาก่อนว่ามีหลักการอะไร พอมีคนพูดถึง ก็จะกลัวไปก่อน คิดว่ายากไปก่อน เนื่องจากไม่มีความเข้าใจในหลักการ แต่หลักจากที่ได้เรียนวิชา image processing ไปแล้ว ความรู้สึกต่างๆก็เริ่มเปลี่ยนไป แม้ว่าจะเป็นการเรียนการสอนแบบ self-study โดยระหว่างขั้นเรียนจะสามารถปรึกษาอาจารย์และ TA ได้เมื่อพบเจอปัญหา โดยทุกคนก็จะให้คำแนะนำที่ดี มีประโยชน์ และชี้แนะในสิ่งที่เราอาจมองข้ามไปหรือคิดว่าไม่สำคัญ

การที่อาจารย์สอนให้เรียนรู้ทุกอย่างตั้งแต่พื้นฐาน เช่นการทำกล้องรูเข็ม การถ่ายภาพ โดยนำวิทยากรรับเชิญมาให้ความรู้เรื่องการถ่ายภาพ ทำให้เกิดความสนใจในด้านนี้มากขึ้น จากนั้นจึงเข้าสู่แก่นของวิชา image processing จริงๆ โดยอาจารย์แบ่งนักศึกษาออกเป็นกลุ่มๆ ฝึกให้นักศึกษาได้ทำงานร่วมกันเป็นทีม ช่วยกันแก้ไขปัญหาระหว่างการทำงาน รู้จักการแบ่งเวลา รู้จักความตรงต่อเวลา

ภาวินี จาตุรพิศานุกูล ()

การได้มีโอกาสศึกษาวิชา Image Processing เป็นการสร้างประสบการณ์การเรียนรู้แบบใหม่ที่ไม่คิดว่าจะได้ลอง ระบบการเรียนการสอนนี้ คือ Self - study เพราะเป็นระบบการสอนของต่างประเทศเป็นส่วนใหญ่ ซึ่งโดยส่วนตัวไม่ถนัดการเรียนการสอนรูปแบบนี้ แต่เนื่องจากมีความสนใจในเนื้อหาวิชา จึงนำเอาวิชานี้ได้มาเป็นแรงผลักดันให้เรียนตามระบบที่อาจารย์ผู้สอนวางไว้ ทำให้ต้องใช้ความพยายามเป็นอย่างมาก แต่ก็ได้รับสิ่งที่ประทับใจ คือ เพื่อนๆ ที่คอยช่วยเหลือ และกลุ่มอาจารย์ผู้สน ที่คอยดูแลและให้คำแนะนำตลอดการสอน

ความรู้สึกต่อเนื้อหารายวิชานี้ คือ รู้สึกแปลกใหม่กับการวางแผนการสอนที่เป็นรูปแบบใหม่ และเนื้อหาการสอนที่ใหม่และแปลกกว่า Image Processing ทั่วไป คือ มีการสอนตั้งแต่พื้นฐานการสร้างภาพ การทำกล้อง ความหมายของศัพท์ที่เกี่ยวข้อง แม้กระทั่งหลักการด้านศิลปะ เช่น การถ่ายภาพ การแต่งภาพ และการวางองค์ประกอบภาพ

รัฐพงศ์ หลอดทอง ()

อย่างแรกเลยการเรียนวิชา image processing เป็นรูปแบบการเรียนรู้ด้วยตัวเองชนิดหนึ่งเป็นการพํฒนาทักษะในการเขียน algorithm จากการเรียนรู้จากนอกห้องเรียนนี้คือ ข้อดีของวิชานี้ ส่วนข้อเสียคือวิธีการการทำงานภายในกลุ่มค่อนข้างจะมีปัญหาอยู่บ้างทำให้กลุ่มของเราสามารถแก้ปัญหาเฉพาะหน้าและแก้ปัญหาต่างๆได้อย่างมืออาชีพ

จิรวัฒน์ เอี่ยมสำอางค์ ()

รูปแบบการสอนในวิชานี้เป็นแบบใหม่สำหรับการเรียนการสอนในมหาวิทยาลัย การเรียนการสอนแบบเน้นผู้เรียนเป็นศูนย์กลางจะทำให้ตัวผู้เรียนได้รับความรู้มากกว่าการเรียนปกติ เพราะความรู้ทีได้จะมาจากการขวนขวาย (Inquiry) ของตัวนักเรียนเอง ไม่ได้มาจากความรู้ของอาจารย์ผู้สอนคนใดคนหนึ่ง

รูปแบบการเรียนแบบนี้กำลังปฏิวัติการเรียนรู้ ในหลายๆมหาวิทยาลัยกำลังมีการเปลี่ยนแปลงไป เช่นการปฏิวัติมหาวิทยาลัยในจีนที่ยกระดับผลงานวิชาการให้สูงขึ้นมาได้ในช่วงเวลา 5 ปีที่ผ่านมา และของเกาหลีใต้เช่น KAIST ที่จะให้นักเรียนอ่านเนื้อหามาเองทั้งหมด ชั้นเรียนจะใช้เป็นเพียงศูนย์การในการ discuss รวมกัน แม้แต่ใน Internet ก็สามารถพบเห็นวิธีการแบบนี้ได้เช่น Cousera Udacity เป็นต้น

อย่างไรก็ตามวิธีการเรียนแบบนี้ไม่ได้ดีที่จะนำมาทดแทนการเรียนการสอนแบบเดิมได้ทั้งหมด บางวิชาการเรียนแบบนี้อาจจะไม่เหมาะสมเช่นวิชาพื้นฐาน ฟิสิกส์ เคมี ชีววิทยา ภาษาศาสตร์ต่างๆ อีกทั้งยังมีข้อกำจัดของตัวผู้เรียนที่มีเวลาจำกัด ทำให้เมื่อเรียนวิชาที่สอนแบบนี้พร้อมกันหลายๆวิชา จะทำให้ผู้เรียนเหนื่อย เนื่องจากการขวนขวายที่มากเกิน อีกทั้งการให้การตำราเรียนมาก่อนล่วงหน้าทำให้เวลาของนักเรียนที่จะไปทำกิจกรรมในด้านอื่นๆลดลง ต้องมาลงให้กับการเรียนแต่เพียงอย่างเดียว

ดังนั้นการเรียนการสอนแบบนี้จะเห็นว่ามีสิ่งที่จะต้องพิจารณาก่อนจะนำมาใช้งานเรียกได้ว่าเป็น “กำแพงที่มองไม่เห็น” ที่จะกั้นไม่ให้ได้ประสิทธิภาพที่ดีที่สุด ถ้ากำแพงด้านเวลาในชีวิต และกำแพงด้านข้อกำจัดของแต่วิชา สามารถทลายลงได้การเรียนการสอนแบบนี้จะทำให้ตัวเรียนได้รับความรู้คุ้มค่าที่สุดเมื่อเทียบกับการเรียนรู้แบบเดิมๆ

สำหรับการเรียนวิชานี้ ความรู้สึกที่เกิดขึ้นตลอดคือความตื่นเต้น ไหนจะตื่นเต้นตอนเช้าวันจันทร์ที่จะมี Quiz สัปดาห์ที่ผ่านมา ไหนจะตื่นเต้นตอนเที่ยงของวันจันทร์ที่จะต้องส่งงานให้ทันภายใน 11.00am และตื่นเต้นอีกทีตอนเที่ยงของวันอาทิตย์ที่จะต้องปั่นรายงานให้อกมาบน website

ขอขอบคุณอาจารย์และพี่ๆทุกท่านที่ทำให้การเรียนวิชามีทั้งความสนุก รื่นเริง แต่ก็ยังเนื้อหาที่อัดเน่น ขอขอบคุณครับ
